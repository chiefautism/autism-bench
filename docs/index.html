How it works

The model receives N constraints and must write a single sentence that satisfies all of them simultaneously. Every constraint is verified by code — no LLM judge.

Scoring

+1 per satisfied constraint. ×2 bonus if all constraints pass (perfect solve).

Validity = constraints passed / constraints attempted.

Perfect solve rate = tasks where every constraint passed.

Constraint categories

  • Structural — word count, char count, word length at position
  • Lexical — must contain color / animal / profession / digit
  • Positional — first word length, last word suffix, starts with letter
  • Relational — unique first letters, ascending length, acrostic
  • Meta — vowel count, unique letters, word length sum, forbidden letter
